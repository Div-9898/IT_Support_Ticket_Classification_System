# IT Support Ticket Classification System

This project provides an automated system for classifying IT support tickets using machine learning and natural language processing techniques.

## Project Structure

```
it-support-ticket-classification-system/
├── frontend/           # Frontend application (React/Vue/Angular)
├── backend/            # Backend API and ML services
├── assets/             # Shared assets (images, fonts, static files)
├── docs/               # Documentation and project resources
└── README.md           # This file
```

## Directory Overview

- **frontend/**: Contains the web application interface for submitting and viewing classified tickets
- **backend/**: Houses the API server, machine learning models, and data processing services
- **assets/**: Stores shared resources like images, fonts, and other static files
- **docs/**: Contains project documentation, API specs, and development guides

## Getting Started

Please refer to the README files in each subdirectory for specific setup instructions.

## Development

Each component of the system is designed to be developed and deployed independently while maintaining clear interfaces for communication.
